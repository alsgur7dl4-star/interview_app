# AI 면접 코치 웹앱

## 프로젝트 개요

8주차에 만든 CLI 면접 코치를 Streamlit 프론트엔드 + FastAPI 백엔드 웹앱으로 확장한 프로젝트입니다.
이력서 기반 맞춤 질문 생성, SSE 스트리밍 면접 응답, 면접·이력서·설정 멀티페이지 구조를 제공합니다.
사용자는 웹 화면에서 면접 연습과 이력서 기반 질문 생성을 확인할 수 있으며, report.py에는 대화 내역을 마크다운 리포트로 만드는 함수와 다운로드 버튼 함수 골격을 구현했습니다.

## 실행 방법

1. uv 가상환경(.venv)을 준비하고 의존성을 설치합니다.

   ```powershell
   uv sync
   ```

2. FastAPI 백엔드를 8002번 포트에서 실행합니다.

   ```powershell
   uv run uvicorn backend.main:app --reload --port 8002
   ```

3. Streamlit 프론트를 8502번 포트에서 실행합니다. (새 터미널)

   ```powershell
   uv run streamlit run frontend/app.py --server.port 8502
   ```

4. API 키는 `.env` 파일에 설정합니다. (`.env` 파일은 git에 올리지 않습니다.)

백엔드가 꺼져 있을 때: 에러 핸들링 함수는 utils.py에 구현되어 있으며, 화면 연결 시 연결 실패 안내를 표시하도록 구성했습니다. uvicorn 실행 여부와 8002번 포트를 확인하세요.

## 핵심 기능 5개

1. Streamlit 기반 면접 코치 채팅 UI
2. FastAPI SSE 기반 면접 응답 스트리밍
3. 멀티에이전트 면접 / 이력서 기반 맞춤 질문 생성
4. 면접 연습 · 이력서 분석 · 설정 멀티페이지 구조
5. 면접 리포트 생성 함수와 다운로드 버튼 함수 골격, 로딩·에러·피드백·검색 UX 보조 기능

## 기술 스택

- Python 3.11+
- Streamlit >= 1.58.0
- FastAPI >= 0.136.3
- httpx >= 0.28.1
- pandas (Streamlit 의존성으로 포함, 3.0.3 확인)
- openai-agents >= 0.17.4 (Agents SDK)
- openai >= 2.41.0 / anthropic >= 0.107.1
- uvicorn >= 0.49.0
- uv (패키지·가상환경 관리)
- Windows PowerShell 실행 환경

환경 확인: 설정 페이지에서 모델 선택 값을 관리하며, 실제 API 호출은 현재 연결된 백엔드 설정을 따릅니다.

## Day 1~5 완성 과정

- Day 1: Streamlit 기본 면접 화면과 역할 선택 흐름
- Day 2: FastAPI SSE 백엔드와 세션 API
- Day 3: 프론트 API 클라이언트와 Agents SDK SSE 연결
- Day 4: 멀티페이지 구조와 이력서 기반 질문 생성/대시보드
- Day 5: utils.py(로딩·에러·피드백·검색), report.py(세션 관리·대시보드·리포트), README, 최종 제출 준비
